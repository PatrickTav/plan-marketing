
# Teste plan  Marketing
## Começando

Primeiro, para baixar:

```bash
git clone "(Link do repositório)"
```
entre na pasta plan-marketing:

```bash
cd plan-marketing
```
Para instalar todas as dependências:
```bash
npm install
```


para rodar a aplicação:
```bash
npm run dev
```
